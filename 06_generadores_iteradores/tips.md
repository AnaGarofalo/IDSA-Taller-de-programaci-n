# Tips para los ejercicios

## Ejercicio simple (pares_hasta)

Usá `yield` dentro de un loop. Recordá que `range(0, n+1, 2)` genera números de 0 a n de 2 en 2.

## Ejercicio complejo (en_chunks)

Usá slicing (`lista[inicio:fin]`) para obtener cada bloque. Avanzá de a `tamanio` posiciones en cada iteración.
