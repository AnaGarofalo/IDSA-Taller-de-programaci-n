# Tips para los ejercicios

## Ejercicio simple (cuadrados_pares)

Combiná filtro y transformación en una sola comprehension:
```python
[expresion for x in lista if condicion]
```

## Ejercicio complejo (procesar_datos)

Podés resolverlo con una list comprehension que tenga:
- Una expresión que cree el nuevo diccionario
- Una condición que filtre por stock
