# Tips para los ejercicios

## Ejercicio simple (bubble_sort)

Recordá que para intercambiar dos elementos en Python podés hacer:
```python
lista[i], lista[j] = lista[j], lista[i]
```

## Ejercicio complejo (ordenar_registros)

Podés usar el algoritmo de bubble sort pero recomiendo usar selection sort. En lugar de comparar números directamente, compará `diccionario[clave]`.
